# Recruitment Backend

## Description

Backend service for the HR recruitment system.

## Project Setup

```bash
$ npm install
```

## Development

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testing

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
